# To Do List App
## Overview
This was a project to practice JavaScript and create a dynamic web app.
Link to [live version](https://3daddict.github.io/todo-list-js/).

## Feature List
  * Mobile responsive with Pure CSS.
  * Sortable List Items
  * Toggle Side Nav.
  * jQuery UI.
  * Edit and Delete Functionality.
